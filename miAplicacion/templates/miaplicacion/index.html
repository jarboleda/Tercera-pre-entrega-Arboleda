Acerca de ésta página

En ésta página se pueden visualizar datos registrados y tiene formularios para el registro de información de Grupos, Supervisores y Usuarios. Puede usar los links superiores o las listas de abajo para acceder a las páginas internas.

El orden de registro de datos es según se indica en la lista:

  • 1.- Ver lista y agregar Grupos : GRUPOS
  • 2.- Ver lista y agregar Supervisores : SUPERVISORES
  • 3.- Ver lista y agregar Usuarios : USUARIOS
